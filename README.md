# Customer Segmentation & Recommendation System

## Description
This project uses machine learning to segment customers and provide recommendations.

## Features
- Customer segmentation using clustering
- Recommendation system using association rules
- Interactive Streamlit web app

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- MLxtend

## How to Run
1. python train.py
2. python -m streamlit run app.py
